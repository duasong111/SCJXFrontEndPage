# 首创居学工作室前端页面（微信小程序）

第一步：使用git去对项目进行拉取

$`git clone https://github.com/duasong111/SCJXFrontEndPage.git`

打开终端 输入 :

`$ npm i` 

注意:一定要去将后台数据跑起来，要不然项目显示会存在问题，页面会存在严重的缺失
